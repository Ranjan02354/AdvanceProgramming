# Assignment 19 – Digital Counter and Theme Toggle App (React Native)

## Question

Build a single-screen mobile application using React Native.

The application must include:

* Increment Button
* Decrement Button
* Reset Button
* Theme Toggle Button

Requirements:

* Counter starts at 0
* Counter cannot go below 0
* Light Mode
* Dark Mode
* Dynamic styling using useState

Use:

* React Native
* useState Hook
* TouchableOpacity
* Flexbox Layout

---

## Objective

To develop a React Native application demonstrating state management, event handling, dynamic styling, and responsive mobile UI design.
