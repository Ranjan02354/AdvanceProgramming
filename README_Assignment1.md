# Assignment 1 – Time Complexity Analysis in C

## Question

Write a C program to analyze time complexities in constant time, linear time and quadratic time. Increase the size of input to check the time consumed.

---

## Objective

To analyze and compare the execution time of algorithms having different time complexities:

* Constant Time – O(1)
* Linear Time – O(n)
* Quadratic Time – O(n²)

by increasing the input size and measuring the time taken for execution.

---

## Concepts Used

* Big O Notation
* Time Complexity Analysis
* Function Pointers
* Performance Measurement using `clock()`
* Nested Loops

---

## Files Included

* Assignment1.c


## Expected Output

The program displays a table showing execution times for:

* O(1)
* O(n)
* O(n²)

for different input sizes.

Example:

```text
Input Size      O(1)        O(n)        O(n²)
100
500
1000
2000
4000
```

---

## Time Complexity

| Function         | Complexity |
| ---------------- | ---------- |
| constant_time()  | O(1)       |
| linear_time()    | O(n)       |
| quadratic_time() | O(n²)      |
